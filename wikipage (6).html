Browsing common folders

ImapX provides direct access to all common folders like Inbox, Drafs, Trash and other. Please consider that it is not always possible to determine these folders, as not all server provide the necessary information (folder flags, names).

/* .. Init client, authenticate .. */          
client.Folders.All
client.Folders.Archive
client.Folders.Inbox
client.Folders.Drafts
client.Folders.Important
client.Folders.Flagged
client.Folders.Junk
client.Folders.Sent
client.Folders.Trash

In case you need to access other folders or common folders have not been initialized, simply enumerate through the client.Folders collection or use index or key based access:

/* .. Init client, authenticate .. */          
client.Folders[0]
client.Folders["MyFolder"]
