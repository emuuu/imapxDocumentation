(Un)Marking the message as read (seen)

(Un)Marking a message as read/seen is very simple. Just change the value of Message.Seen property:

/* .. message = any message .. */  
if(message.Seen)
{
     message.Seen = false; // Mark message as unseen
}
else
{
     message.Seen = true; // Mark message as seen
}
