Copying messages

ImapX allows you to copy messages between folders. In order to copy the message, use the Message.CopyTo method

/* .. Init client, authenticate .. */        
/* .. folder = any folder .. */  
/* .. message = any message .. */  
message.CopyTo(folder)

If you want to the client to download the newly created copy of the message in the target folder, set the second argument to true. This is only possible if your server supports the UIDPLUS extension!

/* .. Init client, authenticate .. */        
/* .. folder = any folder .. */  
/* .. message = any message .. */  
message.CopyTo(folder, true)
